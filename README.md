# One Buffalo Prints

Welcome to **One Buffalo Prints**, the open-source 3D modeling playground of **[One Buffalo Labs](https://onebuffalolabs.com)**. 

Forget the promise of "consistent" updates‚Äîmy output here is a lot like a real weather system off Lake Erie: incredibly bursty and highly unpredictable. ## üõ†Ô∏è The OpenSCAD Philosophy

Sharing just an `.stl` file is like sharing a baked cake without the recipe.

By providing the original OpenSCAD (`.scad`) files right here in this repository, you have the power to tweak dimensions, adjust tolerances, and completely remix these designs to fit your exact hardware and needs.

## üìÅ How This Repo is Organized

To keep things clean, every model gets its own dedicated folder. Inside a typical project folder, you'll find:

- `model_name.scad` ‚Äî The original, customizable OpenSCAD file.
- `model_name.stl` ‚Äî The standard exported 3D mesh.
- `images/` ‚Äî Renders and photos of the final printed object.
- `README.md` ‚Äî Specific print settings, required hardware, and assembly guides for that exact model.
